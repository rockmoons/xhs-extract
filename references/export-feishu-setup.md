# xhs-extract · 飞书导出配置教程

> 小红书的飞书导出配置流程与抖音版完全一致。
> 如果你已经配置过 dy-extract 的飞书，直接复制那份配置即可。

---

## 一步步教你配置飞书导出（新手友好）

> 别被"企业"两个字吓到，个人也能用，免费的。

### 第一步：打开飞书开发者后台

访问 https://open.feishu.cn/app

点击 **「创建企业自建应用」**。

### 第二步：填写应用信息

- 应用名称：`xhs-extract`（随便填）
- 应用描述：小红书笔记导出工具
- 其他按默认

### 第三步：开启权限

在应用管理页面 → **「权限管理」**：

搜索并开启以下权限：

| 权限 | 为什么需要 |
|------|-----------|
| `bitable:app` | 创建和写入多维表格 |
| `drive:drive` | 在云空间创建文件 |

全部打勾后点击「批量开通」。

### 第四步：获取凭证

左侧菜单 → **「凭证与基础信息」**：

你会看到：
- **App ID** → 复制，填到 config.json 的 `export.feishu.app_id`
- **App Secret** → 复制，填到 `export.feishu.app_secret`

### 第五步：发布应用

左侧菜单 → **「版本管理与发布」**：

1. 点击「创建版本」
2. 版本号填 `1.0.0`
3. 随便写点更新说明
4. 点击「保存」
5. 点击「申请发布」→ 弹出权限确认 → 确认

> 企业管理员可能需要审批，自己给自己审批就行。

### 第六步：验证

填好 config.json 后，在聊天中说 **「导出飞书」**。

首次会自动创建多维表格，后续数据写入同一张表。

---

## config.json 填写位置

```json
{
  "export": {
    "feishu": {
      "app_id": "你的 App ID",
      "app_secret": "你的 App Secret",
      "bitable_id": "",   ← 留空，首次自动创建
      "table_id": ""      ← 留空，首次自动创建
    }
  }
}
```

`bitable_id` 和 `table_id` 留空，首次导出时会自动创建并回填。

## 常见问题

**Q: 提示"权限不足"？**
A: 检查是否开启了 `bitable:app` 和 `drive:drive` 权限，并且已发布最新版本。

**Q: 创建表格失败？**
A: 检查 App ID 和 App Secret 是否正确，以及网络是否能访问 `open.feishu.cn`。

**Q: 视频笔记和图文笔记字段不一样？**
A: 自动处理——图文笔记的「@用户」「背景音乐ID」为空，视频笔记的「作品图片」「图片数量」可能为空。Excel 里对应格子留空。
