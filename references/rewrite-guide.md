# xhs-extract · AI 改写指南

## 6 种改写模式

| 模式 | 适用场景 | 说明 |
|------|---------|------|
| ① 去重优化 | 原文啰嗦 | 删除口语词、重复表达，保留核心信息 |
| ② 总结精简 | 需要短版 | 压缩到 15秒/30秒/60秒 版本 |
| ③ 扩写丰富 | 内容单薄 | 补充细节、案例、数据 |
| ④ 转风格 | 换平台 | 小红书风 / 新闻风 / 幽默 / 深度 / 励志 / 悬念 |
| ⑤ 提取金句 | 做标题/封面 | 摘出 1-3 句核心观点 |
| ⑥ 拆分多条 | 内容太杂 | 按话题拆分为多个独立段落 |

## 交互规则

### 用户明确说了模式

直接按指定模式改写。

### 用户只说"改写"没指定模式

展示 6 种模式让用户选：

```
想怎么改？

1. 去重优化 — 删掉啰嗦重复
2. 总结精简 — 缩短到15/30/60秒版本
3. 扩写丰富 — 补充细节和案例
4. 转风格 — 小红书风/新闻风/幽默/深度/励志/悬念
5. 提取金句 — 抓出核心观点
6. 拆分多条 — 按话题拆开

回复数字或模式名～
```

### 输出格式

展示原文 + 改写后对比：

```
📝 原文：
{原作品简介}

✏️ 改写后（{模式名}）：
{改写结果}

💡 不满意可以说「换一种」或指定其他模式
```

### 字段关联

改写完成后，将结果关联到当前笔记数据，以便导出时包含：

- `改写文案`: 改写后的内容
- `改写模式`: 使用的模式名
